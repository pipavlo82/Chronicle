# Non-Goals

Chronicle is not trying to be:

- another NFT project
- another token launch
- another marketplace
- another wallet-only ownership scheme
- a smart-contract implementation repo
- a blockchain selection exercise
- a speculation-first asset wrapper

Chronicle also does not attempt, in this repository, to:

- implement settlement logic
- implement royalties
- implement a marketplace backend
- define a final chain-specific standard
- replace proof systems like ReceiptOS
- replace work/settlement systems like CYPHES

Chronicle is a research project about whether receipts themselves deserve to be treated as a new digital asset class.
